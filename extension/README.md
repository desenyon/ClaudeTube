# ClaudeTube

**YouTube in your Cursor sidebar** — watch tutorials while you code with Claude.

## Install

Search **ClaudeTube** in the Extensions view, or install from [GitHub Releases](https://github.com/desenyon/ClaudeTube/releases).

## Quick start

1. Click the **ClaudeTube** icon in the activity bar
2. Paste a YouTube URL and hit **Play**

## Features

- Sidebar YouTube player with Compact, Theater, and Mini layouts
- Queue, history, shuffle, and repeat
- Thumbnails and auto-fetched titles
- Status bar now-playing indicator
- Click YouTube links in your editor to play
- Full agent API via `.claudetube/inbox.json`

## Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+Y` | Play / pause |
| `Cmd+Shift+0` | Show player |
| `Cmd+Shift+.` | Next in queue |
